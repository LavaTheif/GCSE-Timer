This webpage is designed to time a GCSE exam. It keeps track of the current time, the time the exam started
and the time that they will need to finish. It also adds on 25% extra time for the students that require
more time to complete the exam than the others. All that is entered by the exams officer is the length of
the exam and the name of it. This can be done for up to 4 exams at once which saves time if you are trying
to make sure all the times are correct and the pupils arnt losing or gaining any random time from a slight
mis-calculation.

It supports formatting for the name of the exam, for example typing '<b>Maths</b>' will result in the
title becoming bold, 'Maths'. Other formats include, italics, <i>, underline, <u>, and marked, <mark>,
and a couple more which can be found here. With the <mark> tag can, the style can be customised
so that the colours are different. This can be done using the following tag: <mark style = "background-color: blue; color: green;">
where blue and green can be other colours. This allows many combinations for colours of backgrounds and text
for the title. Please note. If you do not close the tag, for example by typing </i> for italics,
then the effect will be applied to the times below.

Thank you for using this site and hopefully it works to your standards! Any issues? Help improve the timer by clicking here!
